Automated Bubble‑Rate Monitoring System

A Raspberry‑Pi powered monitoring and valve‑control system that tracks bubble rates during plastination using computer vision, stepper control, and a secure web interface.

Explore Features View Architecture

Computer Vision

Real‑time bubble detection and rate estimation with camera input.

Automated Control

Stepper‑driven needle valve adjustments based on target setpoints.

Secure Web UI

Live telemetry, charts, and controls accessible over the network.

Data Logging

Session history, alerts, and export for analysis and reporting.

System Architecture

  • Raspberry Pi captures video and runs CV pipeline.
  • Motor Driver actuates stepper for valve control.
  • Web Server publishes metrics and controls securely.
  • Storage persists time series for later review.
